 Selenium using Java

Selenium can be used using Java language. Basically we just need to import selenium classes and webdriver in our IDE(Eclipse,IntelliJ etc). There are some classes defined in those selenium frameworks which helps us to gain a full control on any webBrowser namely Mozilla,Chrome,Edge and even Internet Explorer.



 Setup in Ecllipse IDE

Now let us look how create a basic Selenium Project in an ecllipse IDE.

Create a new project as mentioned in Screenshot


Select Java Project and Click Next


Write a suitable name for your project and click next


Now, here comes the significant part. Go on libraries and add which Selenium Library you want to add. I am using Selenium 3.4 library


Click on Finish and bang!! You are ready to go and control the browsers
